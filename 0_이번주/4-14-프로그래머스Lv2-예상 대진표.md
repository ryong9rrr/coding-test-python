# 접근 1 : 이분탐색 + 수학

"올림" 연산을 이용해서 라운드를 계산하고 비교하는 방법

#### python

```python
import math

def get_round(num):
    return math.ceil(num / 2)

def solution(n,a,b):
    ans = 1
    while get_round(a) != get_round(b):
        a = get_round(a)
        b = get_round(b)
        ans += 1
    return ans
```
